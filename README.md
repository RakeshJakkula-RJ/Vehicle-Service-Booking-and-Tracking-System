# Vehicle Service Booking & Tracking System

## Description:
The **Vehicle Service Booking & Tracking System** is a full stack web application designed to simplify the process of booking vehicle services and tracking their progress in real time.  
Users can book service appointments, view service history, and track the current status of their vehicle, while service providers can manage bookings and update service statuses.

This project is built using **React.js** for the frontend and **Spring Boot** for the backend, following a RESTful architecture.

---

## Features:
- Vehicle service booking
- Service status tracking (Booked, In Progress, Completed)
- View service history
- Responsive and user-friendly UI
- REST API based backend
- Secure and scalable architecture

## Installation & Setup:

### Prerequisites
- Node.js
- Java (JDK 17 or above)
- Maven
- MySQL / MongoDB

---

### Frontend Setup

cd frontend
npm install
npm start
---
### Backend Setup
cd backend
mvn clean install
mvn spring-boot:run
